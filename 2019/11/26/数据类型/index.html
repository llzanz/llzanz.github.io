python有无数种数据类型

python有六大类标准数据类型：

  • Number 数字类型
    1. Int 整型
    2. Float 浮点型
    3. Bool 布尔型
    4. Complex 复数

下面这些是容器类数据

  • String 字符串类型
  • List 列表类型
  • Tuple 元组类型
  • Set 集合类型
  • Dict 字典类型

声明一个二进制变量0~1：intvar = 0b10101

声明一个八进制变量0~7：intvar = 0o247

声明一个十六进制变量09 AF：intvar = 0xd18f

声明复数complex(5, 6)输出5+6j4

常见转译字符

\n 表示换行符号

\r 表示回车符号

\ 续航符号

\t 缩进符号

在字符串前加r或R 不转译字符串里任何转译字符

四种简单介绍

List[] 有序

Tuple() 有序 不可修改内容

Set{} 无序 不重复

Dict{键:值,键:值} 键值对

数据类型转换

分类：自动数据类型转换（隐式）和强制数据类型转换（显式）

自动数据类型转换：程序自发的行为，在需要的情况下进行的操作，无法干预

强制数据类型转换

根据程序员的开发意愿，自己书写的代码进行的操作

强制转向为整型：int()

浮点型转换，去掉小数部分

布尔值转换，True为1、False为0

复数不能转换

只有整数内容的字符串能转换

列表、元组、字典、集合都不能转

强制转换为浮点型：float()

整型转换，添加.0

布尔值转换，True为1.0、False为0.0

复数不能转换

是数值类型的字符串就可以转换

列表、元组、字典、集合都不能转

强制转换为布尔型：bool()

整型0为False

浮点型0.0为False

复数0j为False

空字符串为False

空的列表、元组、字典、集合为False

强制转换为复数：complex()

整型、浮点型转换，后面加0j

布尔值True为1+0j，False为0+0j

纯数值字符串可以转换

列表、元组、字典、集合都不能转

强制转换为字符串：str()

内容不变，格式转变

只有容器类数据才能转换为容器类数据，字符串除外

强制转换为列表：list()

字符串转换，把里面每一个字符转换成每个值

元组转，内容不变，改变类型

集合转，内容不变，改变类型，顺序随机

字典转，只保留键，舍弃值

强制转换为元组：tuple()

字符串转换，把里面每一个字符转换成每个值

列表转，内容不变，改变类型

集合转，内容不变，改变类型，顺序随机

字典转，只保留键，舍弃值

强制转换为集合：set()

字符串转换，去掉重复字符，剩下字符变成每个值，顺序随机

列表，元组转换，去重，随机

字典转换，留键，去值

强制转换为字典：dict()

字符串不能转换

列表、元组、集合普通不能转，特殊格式能转
