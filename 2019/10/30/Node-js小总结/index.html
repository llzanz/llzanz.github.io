Node.js小总结

2019-10-30
笔记
  • Node.js
本文总阅读量次

Node.js是什么

  • JavaScript 运行时
  • 既不是语言，也不是框架，他是一个平台

Node.js中的JavaScript

  • 没有BOM、DOM

  • EcmaScript基本的JavaScript语言部分

  • 子Node中为JavaScript提供了一些服务器级别的API

    • 文件操作的能力
    • http服务的能力

    模块系统

  • 在Node中没有全局作用域的概念

  • 在Node中，只能通过require方法来加载执行的多个JS文件

  • require加载只能执行其中的代码，但是某些情况下，模块与模块需要进行通信，因此在每个模块中，都提供了一个对象exports，该对象默认是一个空对象

  • 需要把想要被外部访问的成员手动挂载到exports上，require这个对象就可以得到exports中的对象。

核心模块

  • 核心模块是又Node提供的一个个具名的模块，他都有自己特殊的名称标识，例如

    • fs 文件操作模块
    • http 网络服务构建模块
    • os 操作系统信息模块
    • path 路径处理模块

  • 核心模块需要手动用require加载，var fs = require('fs')
