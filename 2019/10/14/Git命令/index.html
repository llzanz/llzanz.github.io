Git命令

2019-10-14
笔记
  • Git
本文总阅读量次

安装git

Linux平台：sudo apt-get install git

设置:

git config --global user.name "Your Name"

git config --global user.email "email@example.com"

创建版本库：git init

添加文件到仓库

添加单个文件：git add

添加全部：git add .

对提交进行说明：git commit -m ""

查看当前状态：git status

查看不同

查看工作区和暂存区的区别：get diff

查看工作区和版本库的区别：grt diff HEAD -- 文件名

查看分支不同：git diff 原分支 目标分支

时光穿梭机

查看历史版本

git log

git log --pretty=oneline

退回版本

git reset --hard HEAD^

git reset --hard 版本号

查看历史命令：git reflog

整理版本：git rebase

用版本库的文件替换工作区的文件：git checkout -- 文件名

撤销add了的文件： git reset HRAD 文件名

删除文件：git rm 文件名

上传本地到远程：git push origin master

关联：git remote add origin git@server-name:path/repo-name.git

下载远程到本地：git clone

分支

查看分支：git branch

创建分支：git branch 分支名

切换分支：git checkout 分知名

创建并切换分支：git checkout -b 分支名

合并分支到当前分支：git merge 分支名

禁用快速模式合并分支：git merge -- no-ff -m "" 分支名

删除分支：git branch -d 分支名

强制删除一个没被合并过的分支：git branch -D 分支名

删除远程分支：git push origin --delete 分支名

复制一个提交的修改到某个分支：git cherry-pick 版本号

暂存

进行暂存：git stash

查看暂存列表：git stash list

恢复暂存：git stash apply

删除暂存：git stash drop

恢复并删除暂存：git stash pop标签

多人协作

查看远程仓库信息：git remote/git remote -v

推送分支：git push origin 分支名

设置本地分支和远程分支的链接：git branch --set-upstream-to=origin/dev dev

从远程拉取最新分支到本地：git fetch

从远程拉取最新分支到本地并合并：git pull

标签

创建标签：git tag 标签

默认打在最新提交的commit上

在特定commit上打标签：git tag 标签 版本号

创建带有说明的标签：git tag -a 标签 -m "说明" 版本号

查看标签信息：git show 标签

查看所有标签：git tag

删除标签：git tag -d 标签

创建的标签只存储到本地，不会推送到远程。

推送标签到远程：git push origin 标签

推送所有标签到远程：git push origin --tags

删除远程标签

先删除本地标签：git tag -d 标签

再删除远程标签：git push origin :refs/tags/标签

完
