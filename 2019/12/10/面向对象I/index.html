面向对象

Object Oriented

面向对象的学习：

  • 面向对象的语法
  • 面向对象的思想

面向对象的优势：

  • 能够实现良好程序结构，方便程序的开发和管理维护

效率：

  • 运行效率：面向对象效率不如过程化快
  • 开发效率：面向对象比过程化开发效率高

面向对象的结构特色：

  • 高内聚：具有关联的功能放在一个结构中
  • 低耦合：没有关联的功能放在不同结构中

面向对象相关单词：

  • OO 面向对象
  • OOP 面向对象的程序开发
  • OOA 面向对象的分析
  • OOD 面向对象的设计
  • OOI 面向对象的实现

OOA -> OOD -> OOI 面向对象的实现过程

对象：唯物

类和对象的关系

类是由对象总结而来的。这个过程叫做抽象化

对象是由类具体实施而来。这个过程叫做实例化

类的组成

类中只有2中内容：成员属性和成员方法

声明一个类必须使用class关键字

命名使用大驼峰命名法

类中只能存在两种内容，成员属性和成员方法。

声明成员方法时，按照函数的规则即可

类成员操作

检测类的成员

格式：类名.__dict__

访问类中成员

格式：类名.成员属性名

格式：类名.成员方法名()

修改类中成员

格式：类名.成员属性名 = 新值

格式：类名.成员方法名 = 函数

添加类中成员

格式：类名.新成员属性名 = 值

格式：类名.新成员方法名 = 函数

删除类中的成员

格式：del 类名.成员属性名

格式：del 类名.成员方法名

格式：对象名.__dict__

访问对象中成员

格式：对象名.成员方法名

修改对象中成员

格式：对象名.成员属性 = 新值

格式：对象名.成员方法 = 函数

添加对象中成员

格式：对象名.新成员属性 = 值

格式：对象名.新成员方法 = 值

删除对象中成员

格式：del 对象名.成员属性
注意：只能删除属于当前对象的成员

格式：del 对象名.成员方法名

self

他不是关键字，是一个随意书写的字符串而已

  • 英文单词的意义 ：自己
  • 绑定类的方法，只能通过类来访问的方法，就是绑定类的方法
  • 非绑定类的方法，就是可以通过对象访问的方法就是非绑定的方法
  • 绑定类与非绑定类的方法不取决于语法，而取决于设计者如何设计该方法
  • 一般情况下非绑定类的方法第一个参数写单词self，但是这不是唯一的单词，写什么都行，self不是关键字，不是关键字，不是关键字！
