随机模块

import random

函数

random()

功能：获取0～1之间的随机小数包含0不包含1
格式：random.random()
返回值：浮点数

randrange()

功能：获取指定范围内指定间隔的随机整数数（左闭右开）
格式：random.randrange([开始值,]结束值[,间隔值])
返回值：整数

randint()

功能：获取指定范围内指定间隔的随机整数数（闭区间）
格式：random.randint([开始值,]结束值)
返回值：整数

uniform()

功能：随机获取指定范围内的所有数值包括小数
格式：random.uniform(开始值，结束值)
返回值：随机返回范围内的所有数值（浮点型）

choice()

功能：随机获取列表中的值
格式：random.choice(序列)
返回值：序列中的某个值

shuffle()

功能：随机打乱原列表
格式：random.shuffle(列表)
返回值：None

pickle模块

import pickle

pickle,dumps()

功能：将任意对象序列化成一个bytes
格式：pickle.dumps(数据)
返回值：bytes

pickle.loads()

功能：将bytes返序列化
格式：pickle.loads(bytes)
返回值：数据

pickle.dump()

功能：把对象序列化后写入到文件对象

pickle.load()

功能：把文件对象反序列化为数据
