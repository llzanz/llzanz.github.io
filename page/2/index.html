Linux命令

2019-10-16
笔记
  • Linux

目录和管理文件命令

显示当前目录：pwd

查看当前目录：ls

查看全部文件，包括隐藏：ls -a

查看文件细节：ls -l

查看文件类型：ls -f

改变所在目录

到根目录：cd

到指定目录：cd 文件名

返回上一级目录：cd ..

返回上两极目录：cd ../..

显示文件内容：cat

在文件中查找字符：grep 内容 文件名

创建文件：touch

复制文件：cp

复制文件到目录：cp 文件名 目录

移动文件：mv 文件名 目录

如果覆盖会提示：mv -i

不提示：mv -f

显示文件移动速度：mv -v

删除

删除文件：rm

提示确认删除：rm -i

不提示删除：rm -f

显示删除进度：rm -v

删除某个目录和目录中所有文件：rm -r

删除目录：rmdir

磁盘空间命令

挂载文件系统：mount

卸载已挂载文件：umont

检查各个硬盘分区和已挂载的文件系统磁盘空间：df

显示文件目录和大小：du

检查和修复Linux文件系统：fsck

文件备份和压缩

zip压缩命令：

压缩：zip 压缩文件名 源文件

解压：unzip

有关关机和查看系统信息的命令

关机：shutdown

不是真正关机，只是发出警告：-k

将系统的服务停掉后就重启：-r

将系统的服务停掉后就关机：-h

取消已经在进行的shutdown命令内容：-c

重启：reboot

换机器：网管！

查看目前程序执行的情况：ps

查看目前程序执行的情景和内存使用情况：top

终止一个进程：kill

查看日期：date

设置时间：date -s

查看月历：cal

管理使用者和设立权限的命令

改变权限：chmod

chmod 使用者增减权限 文件名

文件使用者 说明 增减 说明 权限 说明
u 拥有文件的用户 + 添加权限 r 读取权
g 所有者所在的组群 - 删除权限 w 写入权
o 其他人 = 使他成为唯一权限 x 执行权
a 以上全部

删除文件的所有权限：chmod a-rwx 文件名

为文件所有者添加权限：chmod u+rwx 文件名

增加用户：useradd 用户名

修改用户：su 用户名

线上查询的命令

查询一个命令的使用方法：man

定位文件和目录：locate

寻找某个命令的含义：whatis

文件阅读命令

查看文件开头部分：head

查看文件结尾部分：tail

分页查看工具：

方向键移动，q退出：less

空格健和b移动，q推出：more

网络操作命令

传送文件：ftp

结束连线并结束程序：bye

检测主机：ping

远端登录：telnet

远端等入：rlogin

显示网络状态：netstat

定位，查找文件的命令

从path环境变量所列的目录中找出command的位置，并显示完整路径的名称：which

找出特定程序的可执行文件，源代码文件以及manpage的路径：whereis

按条件搜索，并执行一定的动作：find

带记忆的文件搜索：locate

更新slocate的索引数据库：updatedb

其他命令

显示字符串：echo

清除显示其：clear

修改密码：passwd

打印：lpr

查看在打印机列中等待的作业：lpq

取消打印列队中的作业：lprm
